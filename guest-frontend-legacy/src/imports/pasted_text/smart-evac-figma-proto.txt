SMART EVACUATION SYSTEM — FIGMA PROTOTYPE (DARK UI + FULL BACKEND FLOW + ADVANCED ANIMATIONS)

Design a high-fidelity, interactive Figma prototype for a guest evacuation system used in hotels. The design must simulate real backend behavior and provide a seamless emergency experience.

The prototype should feel like a real app, not static screens.

---

## DESIGN STYLE (DARK EMERGENCY UI)

Theme:

* Background: #0B0F14 (deep dark)
* Cards: #121821
* Borders: subtle gray (#1F2937)

Colors:

* Danger (fire): #EF4444 (pulsing glow)
* Warning: #F59E0B
* Safe: #22C55E
* Info: soft blue (#3B82F6)

Typography:

* Bold, high-contrast headings
* Minimal text
* Large readable instructions

---

## VISUAL FEEL

* Futuristic + tactical (like emergency control system)
* Soft neon glow accents
* Subtle grid background
* Depth using shadows and blur

CORE USER FLOW (MUST BE PROTOTYPED)

1. Landing Screen
2. Room Selection
3. Session Initialization
4. Safe Dashboard
5. Emergency Trigger → Evacuation Mode
6. Step-by-step Navigation
7. Help / Reroute / Complete actions

INTERACTION LOGIC (SIMULATE BACKEND)

→ Map moves forward

---

REROUTE:
→ Old path fades

“Safer route found”

HELP:

“Help is on the way”
